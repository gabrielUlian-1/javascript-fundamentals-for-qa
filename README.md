# JavaScript Fundamentals for QA 🚀

## 📌 Sobre o projeto

Este repositório foi criado para armazenar os estudos, exercícios e desafios desenvolvidos durante a Mentoria de Testes de Software 2.0, ministrada por Júlio de Lima.

O objetivo deste material é consolidar os fundamentos da linguagem JavaScript, desenvolvendo uma base sólida em lógica de programação, estruturas de dados, funções e boas práticas de desenvolvimento, conhecimentos essenciais para a construção e manutenção de testes automatizados.

---

## 📚 Conteúdos estudados

- Variáveis e tipos de dados
- Operadores lógicos e matemáticos
- Estruturas condicionais
- Estruturas de repetição
- Funções
- Arrays e objetos
- Manipulação de dados
- Tratamento de erros
- Boas práticas de programação
- Resolução de desafios de lógica

---

## 🛠️ Tecnologias utilizadas

- JavaScript
- Node.js
- Git
- GitHub

---

## 🎯 Objetivo profissional

Como futuro profissional de Quality Assurance (QA), este repositório representa minha evolução em lógica de programação e JavaScript aplicado à automação de testes utilizando ferramentas como Cypress e Selenium.
